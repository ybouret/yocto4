\section{Description}
Let us assume that we have a set of data $\lbrace X_i,Y_i\rbrace_{i\in[1,N]}$ with $X_i<X_{i+1}$.
We want to approximate the data with a polynomial of maximum degree $d$ on each window
defined by $[X_i-a,X_i+b]$.
